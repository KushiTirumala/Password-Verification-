Password Verification App (Java)

A Java console application that validates passwords based on security rules. Ensures users create strong passwords using uppercase, lowercase, numbers, special characters, and minimum length constraints.

Project Overview

This project demonstrates Java programming, regex, and input validation. Users can input passwords, and the program verifies their strength, providing immediate feedback.

Features

Validate password strength in real-time

Enforces minimum 8-character length

Checks for uppercase, lowercase, numeric, and special characters

Provides clear feedback for invalid passwords

Technologies Used

Java 8+

Regular Expressions (Regex)

Console-based input/output
